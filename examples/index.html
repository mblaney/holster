Holster - test examples

This file loads index.js which contains examples that use the Holster API. See the console log for test output.

The tests make two updates to the same data, but they occur in random order. The "species" field is then logged and should match the update that happened last.

The index.js file is loaded as a module, which allows it to import the development build of Holster. You can switch to the production build by changing the import path at the top of the file.

See user.html for examples that test the Holster.user API.

See wire.html for examples that test the Holster.wire API.
