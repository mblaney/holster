Holster.wire - test examples

This file loads wire.js which contains examples that use the Holster.wire API. These examples show how conflict resolution works, see the console log for test output.

These tests also make two updates in random order, but the wire API can set state vectors on updates to control which value should be applied. This means when the "species" field is logged it will always be the same no matter which order the tests run in.

The wire.js file is loaded as a module, which allows it to import the development build of Holster. You can switch to the production build by changing the import path at the top of the file.
