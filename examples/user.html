Holster.user - test examples

This file loads user.js which contains examples that test the Holster.user API. See the console log for test output.

The tests create two users, Alice and Bob. They both write data to their own user space. Bob writes secret data that can only be read by Alice, then Alice logs in again to read the message.

These tests also show how to use the SEA API (Security, Encryption, and Authorization), which can be loaded on it's own, but it made available via holster.SEA. The holster.user API uses the SEA API internally for creating and authenticating users.

The user.js file is loaded as a module, which allows it to import the development build of Holster. You can switch to the production build by changing the import path at the top of the file.
